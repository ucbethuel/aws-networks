# AWS Networking Series

My documentation for the **AWS Networking** learning path: nine hands-on projects on Amazon VPC and related concepts. Each folder holds my completed project write-up—what I did, what I learned, and my reflections—in both **markdown (.md)** and **PDF** format.

**Author:** Uchenna Bethel Orji *(ucbethuel)*  
**Email:** orjiucbethuel@gmail.com

---

## What’s in this repo

This repository is my own project documentation. There are **9 folders**, one per project. Inside each folder you’ll find two files: a **.md** (markdown) file and a **.pdf** file—both are my full write-up for that project (steps, screenshots, notes, and reflections). This README points you to each.
